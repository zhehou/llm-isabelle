# Benchmarks

Put one goal per line (no `lemma` prefix needed).  
Run with `python -m prover.bench --suite <name>`.

Suites:
- `lists.txt` – list algebra and rev/map/length basics
- `nat.txt`   – Peano arithmetic identities
- `sets.txt`  – simple set equalities/inclusions
- `logic.txt` – propositional tautologies
